# Nexthak 🚀

Nexthak is a modern mock test platform designed to help learners practice, improve, and prepare for technical interviews, placements, and certifications through realistic mock tests.

## 🌐 Live Demo
https://sarthak-official.github.io/nexthak/

## ✨ Features

- Real exam-like mock test experience
- Timed quizzes
- Aptitude practice tests
- Core Java mock tests
- Cloud/technical quiz support
- Randomized questions
- Performance tracking with results
- Responsive modern UI
- Mobile-friendly design

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- JSON

## 🎯 Purpose

Nexthak helps students and learners:

- Practice mock tests anytime
- Improve problem-solving skills
- Prepare for placements
- Build confidence for real exams

## 🚀 How to Run Locally

1. Clone repository

```bash
git clone https://github.com/sarthak-official/nexthak.git
```

2. Open project folder

```bash
cd nexthak
```

3. Run with Live Server in VS Code
